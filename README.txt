PRADO STABILITY FINTECH - TELA REGISTRO DE GASTO
===============================================

Arquivo principal para abrir: index.html

Este projeto contém uma única tela do app Fintech Prado Stability.
A tela criada é Registro de Gasto, usando HTML, CSS separado e Tailwind CSS.

Pastas:
/css - estilos próprios
/imagens - imagens usadas ou reservadas
/referencias - todas as referecnias do projeto

Para abrir:
Dê dois cliques no index.html.
